# 📖 Project Overview

The **Excel Data Analytics Dashboard** is an advanced Microsoft Excel project that demonstrates practical business data analysis techniques using modern Excel features.

The project focuses on analyzing customer purchase data, identifying business trends, performing statistical analysis, forecasting outcomes, and presenting results through an interactive dashboard.

It combines Excel formulas, Pivot Tables, Conditional Formatting, Data Analysis ToolPak, charts, and What-If Analysis to create an end-to-end analytical solution.

---

# 🎯 Project Objectives

- Analyze customer purchase behavior
- Identify high-value customers
- Perform descriptive statistical analysis
- Predict business outcomes using Linear Regression
- Create an interactive business dashboard
- Visualize business performance
- Present insights for decision making
