# 📖 Overview

**Excel Fundamental Booster** is a practical Microsoft Excel workbook designed to strengthen the fundamentals of Excel through real-world datasets and business scenarios.

Instead of learning formulas individually, this project applies them to multiple datasets including **Student Grades**, **Sales Data**, and **Employee Records**, helping learners understand how Excel functions solve practical business problems.

This workbook is ideal for beginners who want to build a strong foundation before moving to advanced Excel, Power BI, or Data Analytics.

---

# 🎯 Learning Objectives

- Understand essential Excel formulas
- Perform calculations using real datasets
- Learn logical and statistical functions
- Analyze student performance
- Analyze sales data
- Perform employee lookups
- Improve spreadsheet problem-solving skills
